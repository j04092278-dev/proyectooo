Proyecto Destacado: Traductor de Código Morse

Un sistema visual y auditivo en programación

Descripción del Proyecto

Desarrollamos un traductor bidireccional de código Morse que permite convertir texto a señales Morse y viceversa, integrando componentes visuales (LED) y auditivos (buzzer) para una experiencia de aprendizaje completa.

El sistema combina programación en Arduino (C++) para el control hardware y Python para la interfaz gráfica y procesamiento de datos.

Objetivos Principales

  • Facilitar el aprendizaje del código Morse mediante herramientas interactivas
  • Crear un sistema bidireccional de traducción texto-Morse
  • Integrar estímulos visuales y auditivos para mejorar la retención
  • Desarrollar una interfaz intuitiva para usuarios de todos los niveles

Implementación Técnica

Arquitectura del Sistema

El proyecto implementa una arquitectura cliente-servidor donde:

  • Python: Gestiona la interfaz gráfica, procesamiento de texto y comunicación serial
  • Arduino: Controla los componentes hardware y ejecuta las secuencias Morse
  • Serial Communication: Conecta ambos sistemas para intercambio de datos en tiempo real

Funcionalidades Clave

  • Traducción automática texto → Morse
  • Decodificación Morse → texto mediante pulsador
  • Visualización de señales con LED/foco
  • Reproducción auditiva con buzzer
  • Gráficos de tiempos de procesamiento
  • Interfaz intuitiva para aprendizaje
